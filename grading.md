## Submission workflow

Follow the **GitHub Classroom** assignment link on Canvas (“Link to project 2 github assignment”) to fork the assignment. The link creates a repo under your account with instructions and starter code.

1. **Clone** that repo locally: `git clone <link>`
2. **Work** in VS Code (open the file or the whole folder) and complete the assignment per the instructions.
3. **Commit** as you go: `git add <file>` and `git commit -m "<message>"`. Commit **at least four times** while working (e.g., after each new function or method).
4. **Push** to GitHub: `git push`
5. **Submit** a link to your repo on Canvas.

Remember to make **four commits of your own**.
